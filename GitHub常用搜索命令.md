##### 常用搜索命令 （多个命令中间空格隔开）
1. 名称包含  
in:name 关键词   
eg: in:name spring cloud   

2. 描述包含  
in:description 关键词   
eg: in:description spring cloud   

3. readme包含  
in:readme 关键词   
eg: in:readme spring cloud    

4. star数大于等于多少 
stars:>= N 关键词  
eg: stars:>=3000 spring cloud   
between的情况是 N1..N2（其他数据类的同理）  
eg: stars:3000..6000 spring cloud   

5. fork数大于等于多少 
同star

6. 语言  
language:java 关键词  

7. 明确搜索某个人或组织的仓库  
user:关键词  
org:关键词
